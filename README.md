# **Project Overview**

This Power BI dashboard solves the problem of scattered and hard-to-understand personal finance data. It brings transactions, budgets, account balances and savings goals into one place so it is easy to see where money is going, spot overspending, and track savings progress toward important goals like an emergency fund or a new car.

In summary, this dashboard helps to move from financial uncertainty to a state of clarity and control. 

# Data Cleaning and Preparation

üîπ **Data Preparation & Transformation**

- Cleaned and transformed the raw data using Power Query to improve data quality and ensure it was reliable for analysis and reporting.
- Created DAX measures to calculate key financial indicators, including income, expenses, investments, savings progress, savings targets, and account balances across both accounts.

- Selected and applied a consistent colour palette to improve visual accuracy, readability, and overall user experience across the dashboard.
- created a dedicated calendar table to enable accurate time-based analysis and reporting.

# Dashboards and Data Visualisation

**üîπ Data Visualisation**

- KPI cards displaying total income, expenses, and investments, with a monthly breakdown for each metric.
- Column chart comparing budgeted amounts with actual spending across all months.
- Table showing expenses by category and monthly expense breakdown.

- Progress bar showing savings performance, including percentage achieved, amount saved versus target, and due date.
